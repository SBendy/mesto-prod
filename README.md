# Mesto

Сервис для публикации фотографий интересных мест.

## Ссылка на проект

https://sbendy.github.io/mesto-prod/

## Запуск и сборка

```bash
npm install
npm run dev
npm run build
npm run deploy
```

- `npm run dev` — локальный сервер разработки
- `npm run build` — сборка в папку `dist`
- `npm run deploy` — сборка и публикация на GitHub Pages
